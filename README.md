# 다트 & 플러터로 네이버 웹툰 앱 화면 구현

- 사용 스택 : Dart, Flutter
- 에뮬레이터 : 갤럭시 S23 울트라 (안드로이드)
- 기간 : 2023/8/5~8/7


## 기능 :
- 기본 화면에서 오늘의 웹툰을 확인할 수 있으며,
- 특정 웹툰을 클릭하게 되면 해당 웹툰의 썸네일, 줄거리, 장르 등 기본설명과 함께 최신 에피소드들을 확인할 수 있는 화면으로 전환하게 된다.
- 에피소드 목록에 있는 에피소드를 클릭하게 되면 네이버 웹툰페이지로 전환되어 웹툰을 확인할 수 있다.
